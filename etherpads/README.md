All of the etherpads from the [Links to notes](https://github.com/Reproducible-Science-Curriculum/Reproducible-Science-Hackathon-Dec-08-2014/wiki/Links-to-notes-from-meeting) page on the wiki. Mozilla does not guarantee long-term hosting of content on the etherpads. 

The bash script 'get_etherpads.sh' takes a file name, finds links to mozilla-hosted etherpads and downloads them using curl.
